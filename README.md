# Timbuchinger Tap

## How do I install these formulae?

`brew install timbuchinger/tap/<formula>`

Or `brew tap timbuchinger/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "timbuchinger/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
